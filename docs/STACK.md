# Стек

Один VPS, userspace-прокси (без правки ядра/форвардинга), всё ставится скриптами.

## Узлы (что получает пользователь)

| Узел | Протокол | Порт | Транспорт | Зачем |
|------|----------|------|-----------|-------|
| Hysteria | Hysteria2 | 443/udp | QUIC + salamander-obfs | основной, быстрый, маскируется под HTTPS-сайт |
| Hysteria turbo | Hysteria2 | 8444/udp | QUIC без obfs | максимум скорости когда DPI не мешает |
| Reality | VLESS-Reality | 8443/tcp | TLS-перехват чужого SNI | запасной канал, без своего серта |
| XHTTP | VLESS-XHTTP | 443/tcp | реальный TLS за nginx | durable против DPI, выглядит как обычный веб |
| XHTTP+CDN | VLESS-XHTTP | 443/tcp | через Cloudflare | прячет реальный IP/ASN сервера за CF |

Несколько узлов разных протоколов - чтобы при блокировке одного клиент автоматически
переключался на другой. UDP (Hysteria) и TCP (Reality/XHTTP) ломаются разным DPI по-разному.

## Маскировка

- **Hysteria masquerade**: на не-VPN запросы сервер отвечает как реверс-прокси настоящего
  сайта (`www.samsung.com`), активный пробинг видит обычный сайт.
- **Reality**: перехватывает TLS-handshake к реальному `www.samsung.com`, не имеет своего
  сертификата, неотличим от обращения к этому сайту.
- **XHTTP**: настоящий TLS на 443 за nginx, трафик выглядит как HTTPS-веб. Мультиплекс
  размывает паттерн "много хендшейков к одному SNI".
- **uTLS firefox** на всех TLS-узлах: дефолтный chrome-отпечаток Xray палится DPI чаще.

## Маршрутизация (split-tunnel)

Делается на клиенте. RU-хосты (банки, госуслуги, маркетплейсы - `rules/`) идут **напрямую**,
видят домашний IP. Остальное - через VPN. Подробности и форматы - `rules/SOURCES.md`.

## Бот и учёт

- FastAPI backend (127.0.0.1:8090): http-auth для Hysteria, выдача подписки `/s/<token>`,
  кабинет `/u/<token>`, Telegram Mini App `/app`, админка `/admin`.
- aiogram-бот: поток выдачи (трафик/срок/устройства/торренты), `/users`, `/stats`.
- Учёт по таймеру (5 мин): trafficStats Hysteria (9443/9444) + Xray statsquery (10085),
  гашение по квоте/сроку. Лимит устройств по HWID.
- Источник правды - SQLite (`bot/db.sqlite`). Провижинер атомарно перерисовывает Xray-клиентов.

## Тюнинг

`sysctl` BBR+fq, UDP-буферы 32M, backlog, TCP Fast Open, mtu probing - `server/templates/sysctl-99-vpn.conf`.
