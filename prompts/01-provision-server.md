# Промпт 1: поднять VPN-узел

Скопируй текст ниже в AI coding-агента (Claude Code или совместимый), запущенный на сервере
или с SSH-доступом к нему. Замени значения в угловых скобках.

---

Подними VPN-узел из репозитория vpnbanana на моём сервере.

Сервер: `<root@IP или "ты уже на сервере">`
Домен: `<vpn.example.com>` (A-запись уже указывает на сервер, проверь `dig +short <домен>`)

Сделай:
1. Склонируй репозиторий в `/root/vpnbanana` (если уже есть - `git pull`).
2. Проверь, что DNS A-запись домена указывает на IP этого сервера. Если нет - остановись
   и скажи, какую запись добавить.
3. Запусти `DOMAIN=<домен> bash /root/vpnbanana/server/install.sh`. Если порт 443 занят
   другим сервисом на этом боксе - подбери свободные порты и передай через env.
4. Дождись завершения, покажи мне итог: открытые порты, ссылку личной подписки и ссылку
   на список обхода.
5. Если что-то упало - посмотри `systemctl status xray hysteria-server`, `nginx -t`,
   почини причину и запусти install.sh повторно (он идемпотентный).
